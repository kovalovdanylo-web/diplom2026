# Receipt Tracker — Система обліку фіскальних чеків

Дипломний проєкт. Система автоматичного розпізнавання та аналізу українських фіскальних чеків, що складається з двох компонентів: **Telegram-бота** та **веб-застосунку**.

## Можливості

### Telegram-бот
- 📸 Розпізнавання чеків з фото (QR-код → XML ДПС → Claude Vision AI як fallback)
- 🛍 Витягування списку товарів з кожного чеку
- 🏷 Автоматична категоризація витрат (8 категорій)
- 🔍 Підтримка кількох чеків на одному фото
- 🗑 Видалення чеку прямо з повідомлення
- 👤 Авторизація через email + пароль (bcrypt)

### Веб-застосунок
- 🏠 Головна сторінка з дашбордом і швидкою статистикою
- 📋 Таблиця чеків з фільтрами (місяць, категорія) та сортуванням
- 🖼 Галерея фото чеків з лайтбоксом
- 🛒 Перегляд списку товарів по кожному чеку
- 📊 Статистика: кругова та стовпчаста діаграми, топ товарів
- 🔄 Порівняння двох місяців (5 типів візуалізації)
- 🗑 Видалення чеків (по одному або масово)
- 🔑 Авторизація — ті самі акаунти що й у боті

## Технології

| Компонент | Стек |
|-----------|------|
| Бот | Python 3.10+, aiogram 3.7, asyncio |
| Розпізнавання | Claude Vision API (Haiku), pyzbar (QR), BeautifulSoup (XML ДПС) |
| Веб | Flask 3.0, Jinja2, Chart.js |
| База даних | SQLite (aiosqlite / sqlite3), WAL mode |
| Авторизація | bcrypt |

## База даних

| Таблиця | Призначення |
|---------|-------------|
| `users` | Telegram-користувачі |
| `auth` | Акаунти (email + bcrypt-хеш) |
| `receipts` | Чеки |
| `receipt_items` | Товари з чеків |
| `receipt_photos` | Фото чеків (прив'язані до чеку) |
| `api_usage` | Витрати Claude API |

## Швидкий старт

### 1. Встановлення залежностей

```bash
pip install -r requirements.txt
```

### 2. Налаштування

```bash
cp .env.example .env
# Відредагуйте .env — вкажіть BOT_TOKEN та ANTHROPIC_API_KEY
```

### 3. Запуск

```bash
# Бот + веб одночасно:
python run.py

# Або окремо:
python main.py        # тільки бот
python web/app.py     # тільки веб (http://127.0.0.1:5000)
```

## Вартість Claude API

| Режим | Вартість |
|-------|----------|
| QR-код знайдено | ~$0.00003 (тільки категорія) |
| Claude Vision (fallback) | ~$0.002 за чек |

При наявності QR-коду ДПС — Claude використовується лише для визначення категорії ($5 ≈ 150 000 сканувань). Повне розпізнавання через Claude запускається тільки якщо QR не знайдено.
